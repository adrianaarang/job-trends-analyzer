<h2>🚀 Descripción</h2>

<p>
<code>Job Trends Analyzer</code> es una aplicación web interactiva que analiza tendencias del mercado laboral utilizando datos en tiempo real obtenidos desde APIs públicas.
</p>

<p>
Permite explorar ofertas de empleo, identificar patrones de contratación y detectar automáticamente las tecnologías más demandadas.
</p>

---

<h2>🎯 Problema que resuelve</h2>

<ul>
  <li>Dificultad para entender tendencias reales del mercado laboral</li>
  <li>Información dispersa en múltiples plataformas</li>
  <li>Falta de herramientas accesibles para análisis de skills demandadas</li>
</ul>

<h3>👨‍💻 Para desarrolladores</h3>
<ul>
  <li>Identificar tecnologías más demandadas</li>
  <li>Analizar oportunidades laborales por categoría</li>
  <li>Explorar tendencias actuales del mercado</li>
</ul>

<h3>📊 Para analistas</h3>
<ul>
  <li>Visualizar datos en tiempo real</li>
  <li>Extraer insights de descripciones de empleo</li>
  <li>Analizar distribución de empresas y ubicaciones</li>
</ul>

---

<h2>✨ Funcionalidades principales</h2>

<ul>
  <li>🔍 Buscador de ofertas (Python, AI, Data, etc.)</li>
  <li>🎯 Filtro por categoría profesional</li>
  <li>📊 Dashboard interactivo</li>
  <li>🏢 Top empresas contratando</li>
  <li>🌍 Distribución geográfica de empleos</li>
  <li>🛠️ Detección automática de tecnologías</li>
  <li>📈 Visualización de tendencias en tiempo real</li>
</ul>

---

<h2>🌐 Integración con API</h2>

<ul>
  <li>Consumo de datos mediante peticiones HTTP</li>
  <li>Procesamiento en tiempo real con Python</li>
  <li>Transformación a DataFrame con Pandas</li>
  <li>Gestión de errores en peticiones</li>
</ul>

<p>
La aplicación se conecta dinámicamente a APIs públicas para obtener datos actualizados del mercado laboral.
</p>

<ul>
  <li>Uso de Streamlit para desarrollo rápido de dashboards interactivos</li>
